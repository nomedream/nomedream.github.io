Surface Go 3 Run Linux

去年 11 月左右购入。当时兴致冲冲购买好了 TF 卡，准备安装双系统，却在一番检索之后发现微软只在旧版本的苏菲上才支持从 TF 上启动系统，即系统必须挂在 Windows 上才行。
后面 TF 卡就拿去做资料盘了，然后就吃灰了（。
上个月又把他拿出来，准备装双系统。基本思路和网上的双硬盘双系统没什么差别，主要就是把 Linux 的 boot 挂载到 Windows 中。
为什么不整机换成 Linux，那当然是因为这是苏菲啊喂（。

Prepare

  • 苏菲狗 3
  • 有线键盘
  • Windows 和 Linux 镜像

Step

  • 重装 Windows。
    • 苏菲狗 3 出厂自带正版 Windows 11 家庭版。去年刚出的时候玩了一阵子，不就是换皮 Windows 10 嘛，还要搞一堆配置才能适应，于是乎准备装回 Windows 10。
    • 特别说明下，我重装的版本原本是 1909，但这个版本竟然没有合适的驱动……后面还是升级了 Win 10 的版本。驱动官网地址是：Download Surface Go 3 Drivers and Firmware from Official Microsoft Download Center，Win 10 指向的是：Win10_19042_22.023.33209.0。
    • 开机时按住【电源键】和【音量下键】，启动到 BIOS 页面，哦对了，默认的加密记得关掉，然后 BIOS 一通设置，从 USB 启动程序，就可以重装系统了，解锁 BIOS 之后开机上面会有巨丑的红条然后一个开锁标志。
    • 重装过程中，记得给 Linux 系统预留 20G 左右的空间，如果要挂交换分区的话再加个 8G、10G 的。
  • 安装 Linux
    • Linux 采用的是 Ubuntu 22.04。
    • 正常流程，安装时候选其他安装方式。
    • 找到 Windows 那块 SSD，看到之前留出的空间，新建一个 200M 的分区，把 EFI 挂上去，剩下的空间可以都拿去挂 / 也可留 8G 给交换分区，然后留下的再挂 /，记住这里系统(/)一定也要挂载在 SSD 下。
    • home 目录挂载在 TF 卡上，启动选择 Linux 的 EFI 所在分区即可。
    • 正常安装，over。
