Basic

  • 修改镜像源：vim /etc/apt/sources.list
    • ubuntu | 镜像站使用帮助 | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror
  • 系统更新：sudo apt-get update
  • 查看版本号：cat /proc/version（uname -v 、lsb_release -a）
  • 查看所有已安装软件包：apt list --installed
  • 安装 SSH 服务端：sudo apt install openssh-server

  • 查看防火墙状态：sudo ufw status
